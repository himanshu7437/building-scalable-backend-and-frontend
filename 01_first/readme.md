DAYS 1–3: Backend Core Mastery

Goal:
Build a production-grade auth API

You will:

Setup Express server properly

Connect MongoDB

Create User schema

Hash password with bcrypt

Implement JWT access + refresh

Add role-based middleware

Add validation (Joi/Zod)

Central error handling

Add rate limiter

Proper CORS config

How to approach:

First 2 hours:
Understand JWT deeply.

Then design schema on paper.

Then build layer by layer.

Test using Postman.

You must be able to explain login lifecycle step-by-step.
